# 🌱 小潘嚴選商城

> 一站式購物平台 - 保健食品 × 農業產品 × 應援衣服

## 🎯 網站特色

- **💊 保健食品**：天然維他命、深海魚油、益生菌等優質保健產品
- **🌾 農業產品**：苦瓜、小黃瓜、蒜頭、橘子等在地農產
- **👕 應援衣服**：雲林女籃應援T恤、客製化應援用品

## 🌐 線上展示

網站連結：`https://FunFunFunChen.github.io/green-field-mall`

## 🛠️ 技術特色

- 響應式設計（RWD）
- 購物車功能
- 會員系統
- 商品分類篩選
